# VisitorIQ — Visitor Management Platform

A cloud-native visitor management web app. Runs entirely in the browser — no backend server required.

## Live Demo

Deployed via GitHub Pages: `https://YOUR_USERNAME.github.io/visitoriq/`

## Login Credentials

| Field | Value |
|-------|-------|
| Username | `admin` |
| Password | `Admin@1234` |

## Features

- 🔐 Login page with forgot password flow
- 📊 Real-time reception dashboard
- 📋 Full visitor log with CSV export
- ✉️ Pre-registration with QR code
- 🗄️ Database schema viewer (Supabase, Neon, Railway, and more)
- 🚀 Kiosk mode for self-service check-in
- ⚙️ Admin panel: branding, hosts, watchlist, GDPR settings

## Deploy to GitHub Pages

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: **main**, Folder: **/ (root)**
5. Click **Save** — live in ~60 seconds
